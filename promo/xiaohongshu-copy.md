# 小红书推广文案

## 标题

把网易云音乐搬进终端

## 正文

最近做了一个个人向的网易云音乐命令行工具：`ncm-cli`。

它适合经常在终端里工作、又想快速整理音乐库的人：

- `ncm me` 查看当前账号
- `ncm playlist list/show` 浏览歌单和歌曲
- `ncm search song` 搜索歌曲
- `ncm lyric` 直接看歌词
- `ncm playlist add/remove/rename/tags/desc` 管理自己的歌单
- `ncm play` 在 macOS 上调起网易云音乐桌面端播放
- 主要读取命令支持 `--json`，可以接脚本、快捷指令或 Agent

首次使用先执行 `ncm login`，扫码登录后就能把网易云音乐放进自己的终端工作流里。

GitHub：`Davied-H/ncm-cli`

非官方个人工具，不隶属于网易云音乐。播放和版权能力以账号、地区和客户端实际状态为准。

想看安装方法可以留言，我整理在评论区。
