ncm-cli
开源 Go CLI macOS
网易云音乐命令行工具

把网易云音乐
搬进终端

查账号、搜歌曲、看歌词、整理歌单，还能把结果输出成 JSON 给脚本和 Agent 继续处理。

~/music-workflow
$ ncm search song 周杰伦 --limit 3 --json
$ ncm playlist add 123456 210049
$ ncm lyric 210049 # 原始歌词直接读
$ ncm play 210049 # 调起桌面端
歌单管理
创建、加歌、移除、改名、改标签，重度整理党更省时间。
搜索歌词
歌曲、歌单、歌词都能查，适合快速找歌和整理资料。
自动化友好
主要命令支持 --json，脚本、快捷指令和 Agent 都能接。
本地播放
macOS 可通过 ncm play 调起网易云音乐桌面端播放。
适合开发者和音乐库整理用户
首次使用执行 ncm login，扫码登录后即可开始管理自己的网易云音乐数据。
